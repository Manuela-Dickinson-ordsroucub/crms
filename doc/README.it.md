## Docker Compose per lo sviluppo locale

- configurare variabili d'ambiente aggiuntive, se necessario, nel file sottostante


"bash."
docker-compose.yml
```

- Dopo aver configurato gli elementi necessari, esegui il seguente comando:

"bash."
docker-compose up -d
```

Questo creerà le immagini e avvierà i contenitori per frontend, backend e MongoDB.

**_Nota:_** Questa configurazione docker-compose è associata solo allo sviluppo locale.
